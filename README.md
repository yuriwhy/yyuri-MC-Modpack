# YYuri's Minecraft Modpack: MCA Edition

### *My Minecraft modpack focused on adventure*

**Requirements:**
- Minecraft 1.21.1
- Fabric 0.18.1 or later

### Regarding the launcher and which file to download:

*I recommend Prism Launcher for this modpack, but the modpack should work fine with the Curseforge and Modrinth launchers.*

*When downloading the modpack, if you're not using the Curseforge or Modrinth launcher, I recommend downloading the file "prismlauncher-x.x.x-x.zip" as it's the entire modpack, but compressed.*
